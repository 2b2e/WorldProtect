**คุณสมบัติของปลั๊กอิน**<br>
- เป็นปลั๊กอินโพรเทตโลกโดยใช้อีเว้นท์ต่างๆ
- โดยอีเว้นท์ทั้งหมด 22 อีเว้นท์
- place (วางบล็อก)
- break (ทุบบล็อก)
- interact (แตะบล็อก)
- damage (อมตะ)
- pvp (ต่อสู่)
- explode (ระเบิดบล็อกกระจาย)
- weather (ฝนตก)
- item-drop (ทิ้งไอเทม)
- move (เดิน)
- fly (บิน)
- keepinventory (ของตก)
- keepexperience (เอ็กพีตก)
- bucketfill (เทถัง)
- bucketempty (ตักถัง)
- chat (แชท)
- bedenter (นอน)
- eatfood (กินอาหาร)
- food (หลอดอาหาร)
- teleport (วาร์ป)
- itemframe-drop (ไอเทมโชว์ตก)
- banitem (แบนไอเทม)
- bancmd (แบนคำสั่ง)
